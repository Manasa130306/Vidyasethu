# 📘 Vidhyasethu – AI Question Paper Generator

## 🚀 Overview
Vidhyasethu is an AI-powered question paper generation system designed to automatically create question papers based on RBT (Revised Bloom’s Taxonomy) levels. It reduces manual effort, avoids duplication, and ensures balanced question distribution.

---

## 🎯 Features
- 🧠 Automated question generation using NLP (BERT)
- 🔁 Duplicate question detection across multiple sets
- 📊 RBT level-based question classification
- 🤖 Integration of AI agents for efficient workflow
- 🖥️ Simple and user-friendly interface

---

## 🛠️ Tech Stack
- Python  
- BERT (NLP Model)  
- Natural Language Processing (NLP)  
- AI Agents  

---

## ⚙️ How It Works
1. Input question dataset is processed  
2. BERT model classifies questions based on RBT levels  
3. Duplicate questions are filtered  
4. AI agents generate structured question papers  
5. Final output is displayed through a simple interface
